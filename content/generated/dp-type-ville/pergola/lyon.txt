# Pergola à Lyon : Déclaration Préalable Simplifiée

Pour installer une pergola à Lyon, vous devez déposer une déclaration préalable auprès de la mairie de Lyon, qui traite les demandes selon les règles du PLU lyonnais. Ce document urbain définit les conditions d'implantation, les distances aux limites et les hauteurs autorisées dans votre quartier. L'instruction officielle prend un mois, mais les démarches administratives peuvent s'avérer chronophages pour les particuliers.
